## 游戏名：小希的信念
### 一、	项目概述
该项目共有两个选择分支、两个游戏关卡和三个最终结局。  
包含了多选择文字游戏、平台跳跃游戏。  

##### 1.1 游戏核心玩法
1.  平台跳跃和收集：玩家需要在地图上收集7块金币通关
2. 平台跳跃和躲避：玩家需要移动到地图终点，途中有许多移动的病毒，如果碰到病毒则游戏重置。
3. 在剧情的基础上设计了两个选择分支，由玩家的选择来进入不同的游戏页面和结局（详情见流程简介）。
##### 1.2 游戏世界观
游戏建立在真实世界中很多儿童得了癌症后不能痊愈，或治疗时没有受到良好的教育导致无法融入社会的现状。游戏中虚构了小希为患癌儿童，玩家来帮助她选择是否接受教育或能否打败疾病。目的是为了让玩家能真实的体验到癌症儿童的困难从而体谅、帮助癌症儿童。
##### 1.3 游戏美术设计
因为希望能构造一个温暖的故事，所以封面整体呈暖色调，同时用色也非常丰富。在运行项目的过程中，因为我们的项目是以一个孩子“小希”的视角展开的旅途，所以我们希望孩子的视角多增添一些童真和美好，游戏中的各种物件上或多或少都有一定的童趣元素，例如可爱兔子。与此同时，项目整体的上色也偏向水彩的风格以表现出一种相对朦胧的感觉，希望对于孩子来说，他们能走过一段梦幻的旅途。他们的人生能如同游戏中的“小希”一般缓缓亮起。

### 二、游戏指引
##### 2.1 游戏安装运行指引
请先下载*小希的信念_v1.0_Windows*文件夹，然后双击打开.exe文件即可。
##### 2.2 游戏操作指引
通过按下键盘左、右、上来操控角色的移动。
根据关卡开始时文字来完成收集或躲避的操作。
##### 2.3 其他指引或提示
根据选择的不同，有三个不同的结局，可以反复尝试哦。

### 三、游戏现存问题及迭代规划
##### 3.1 现存问题
人物动画不够精美流畅。  
剧情文字比较粗糙。  
页面背景比较单一  
##### 3.2 迭代计划
人物增加更多动画。  
页面增加更多美术设计。  
完善更多剧情，让小希的故事更有感染力，达到更好公益宣传的效果。
